========================================================
  Tor-proxy — доступ к заблокированным сайтам
  by bylka
========================================================

ЧТО ЭТО
  Локальный прокси через сеть Tor (мост Snowflake). Открывает сайты,
  заблокированные провайдером. Это НЕ VPN, программе НЕ нужны права
  администратора и установка — это один файл.
  Скорость ~1–5 Мбит/с: подходит для сайтов, форумов, мессенджеров и
  приложений; для видео — нет.

ТРЕБОВАНИЯ
  Windows 10 или 11 (64-бит) и интернет. Больше ничего не нужно.


--------------------------------------------------------
ЗАПУСК
--------------------------------------------------------
  1. Устанавливать ничего не надо — просто запустите Tor-proxy.exe
     (двойной клик).

  2. При первом запуске Windows может показать синее окно
     «Windows защитила ваш компьютер» (SmartScreen). Это потому, что
     программа подписана самоподписанной подписью (bylka), а не куплен-
     ным сертификатом. Нажмите «Подробнее» → «Выполнить в любом случае».

  3. Антивирус может насторожиться (внутри лежит tor.exe) — это ложное
     срабатывание, разрешите запуск.


--------------------------------------------------------
КАК ПОЛЬЗОВАТЬСЯ
--------------------------------------------------------
  1. Если включён VPN — выключите его.

  2. Нажмите «Запустить». Подождите 1–3 минуты, пока статус не станет
     «Готово». Задержка на ~10% с надписью «Поиск моста» — это нормально,
     идёт подбор рабочего узла Snowflake. Просто ждите.

  3. Нажмите «Копировать» — адрес 127.0.0.1:9150 попадёт в буфер обмена.

  4. Настройте браузер или программу на этот SOCKS5-прокси (см. ниже).

  5. Не закрывайте программу, пока пользуетесь прокси. Окно можно свернуть
     (крестик) — оно уходит в область уведомлений (значок у часов), а
     прокси продолжает работать. Полностью выйти: правый клик по значку в
     трее → «Выход» (или кнопка «Выход» в окне).


--------------------------------------------------------
НАСТРОЙКА БРАУЗЕРА
--------------------------------------------------------
  FIREFOX
    Настройки → «Параметры сети» → «Настроить…» →
    «Ручная настройка прокси»:
      • Узел SOCKS:  127.0.0.1
      • Порт:        9150
      • Вариант:     SOCKS 5
      • Включите «Проксировать DNS при использовании SOCKS 5».
    Нажмите ОК.

  CHROME / EDGE
    Удобнее через расширение FoxyProxy:
      • Тип: SOCKS5, адрес 127.0.0.1, порт 9150, включите «Proxy DNS».
    В FoxyProxy можно задать список сайтов — тогда через Tor пойдут только
    заблокированные адреса, а остальное — напрямую и на полной скорости.


--------------------------------------------------------
НАСТРОЙКА TELEGRAM (на этом же компьютере)
--------------------------------------------------------
  1. Запустите прокси в программе и дождитесь «Готово».
  2. Telegram Desktop → меню ☰ → «Настройки».
  3. «Продвинутые настройки» (Advanced).
  4. «Тип соединения» → «Использовать свой прокси».
  5. «Добавить прокси» → выберите SOCKS5.
  6. Сервер: 127.0.0.1, Порт: 9150. Логин и пароль оставьте пустыми.
  7. «Сохранить» и убедитесь, что этот прокси выбран.

  Примечание: адрес 127.0.0.1 — это «сам компьютер», поэтому прокси видно
  только с этого ПК: так настраивается Telegram Desktop здесь же. Подключить
  к нему телефон нельзя — для телефона есть отдельное приложение, см. раздел
  «ВЕРСИЯ ДЛЯ ТЕЛЕФОНА (ANDROID)» ниже.


--------------------------------------------------------
ПРОВЕРКА
--------------------------------------------------------
  После статуса «Готово» нажмите «Проверить, что работает через Tor» —
  программа откроет check.torproject.org через прокси и покажет, идёт ли
  трафик через сеть Tor и ваш внешний IP.


--------------------------------------------------------
ЕСЛИ НЕ ПОДКЛЮЧАЕТСЯ
--------------------------------------------------------
  • Убедитесь, что VPN выключен и есть интернет.
  • Snowflake иногда долго ищет рабочий узел — подождите пару минут или
    нажмите «Остановить», затем «Запустить» снова.
  • В разделе «Дополнительно» можно добавить свои мосты obfs4 (получить на
    https://bridges.torproject.org или в Telegram у @GetBridgesBot),
    нажать «Сохранить мосты» и перезапустить прокси.


--------------------------------------------------------
ДОПОЛНИТЕЛЬНО
--------------------------------------------------------
  • «Запускать вместе с Windows» — галочка в разделе «Дополнительно».
  • Файлы программы лежат в  %LOCALAPPDATA%\TorProxy
    (кнопка «Открыть папку с настройками»).


--------------------------------------------------------
ВЕРСИЯ ДЛЯ ТЕЛЕФОНА (ANDROID)
--------------------------------------------------------
  Эта инструкция — про версию для Windows. Для телефона есть отдельное
  приложение, оно устроено ИНАЧЕ: не прокси, который надо прописывать
  в каждой программе, а выбор приложений галочками.

  ГДЕ ВЗЯТЬ
    В том же релизе, откуда вы скачали Tor-proxy.exe:
      • Tor-proxy-by-bylka-arm64.apk      (41 МБ) — почти всем телефонам
      • Tor-proxy-by-bylka-universal.apk  (123 МБ) — если первый не ставится

  КАК ПОЛЬЗОВАТЬСЯ
    1. Установите APK (Android спросит разрешение на установку из этого
       источника — разрешите).
    2. Нажмите «Подключить», разрешите запрос VPN и дождитесь
       «Подключено» (те же 1–3 минуты на подбор моста).
    3. Нажмите «Выбор приложений» и отметьте те, которым нужен Tor.

    Через Tor идут ТОЛЬКО отмеченные приложения. Пока не отмечено ни
    одного — через Tor не идёт ничего. Остальной трафик телефона всегда
    остаётся на обычном соединении: весь телефон в Tor не заворачивается.

    Внутри Telegram и браузера настраивать НИЧЕГО не нужно — достаточно
    отметить их в списке. Telegram через Tor может грузиться до нескольких
    минут, это нормально.

    Подробная пошаговая инструкция — во вкладке «Инструкция» внутри
    самого приложения.


--------------------------------------------------------
О ПРИВАТНОСТИ
--------------------------------------------------------
  Трафик идёт через сеть Tor: провайдер не видит, какие сайты вы
  открываете, а сайт не видит ваш настоящий IP. Это обход блокировки,
  а не гарантия полной анонимности.
