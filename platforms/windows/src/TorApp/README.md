========================================================
  Tor-proxy — доступ к заблокированным сайтам
  by bylka
========================================================

ЧТО ЭТО
  Доступ к сайтам и приложениям, заблокированным провайдером, через сеть
  Tor (мост Snowflake). Есть две версии — для компьютера и для телефона.
  Обе сделаны так, чтобы просто работать: скачал, включил, пользуешься.

  [WINDOWS]  Локальный прокси. Это НЕ VPN, установка и права
             администратора не нужны — это один файл. Программы, которым
             нужен Tor, вы направляете на адрес 127.0.0.1:9150.

  [ANDROID]  Приложение с выбором галочками. Через Tor идут ТОЛЬКО
             отмеченные вами приложения, остальной трафик телефона не
             затрагивается. Внутри самих приложений настраивать ничего
             не нужно.

  Скорость у обеих версий ~1–5 Мбит/с: сайты, форумы, мессенджеры и
  приложения — да; видео в высоком качестве — нет.

ТРЕБОВАНИЯ

--------------------------------------------------------
УСТАНОВКА И ЗАПУСК
--------------------------------------------------------
  [WINDOWS]
    1. Устанавливать ничего не надо — просто запустите Tor-proxy.exe
       (двойной клик).

    2. При первом запуске Windows может показать синее окно
       «Windows защитила ваш компьютер» (SmartScreen). Это потому, что
       программа подписана без купленного сертификата. Нажмите
       «Подробнее» → «Выполнить в любом случае».

    3. Антивирус может насторожиться (внутри лежит tor.exe) — это ложное
       срабатывание, разрешите запуск.

  [ANDROID]
    1. Откройте скачанный APK. Android спросит разрешение на установку
       приложений из этого источника — разрешите.

    2. При первом подключении Android покажет «Запрос на подключение»
       (запрос VPN) — разрешите. Это локальный VPN на самом телефоне,
       ваш трафик не передаётся третьим лицам.


--------------------------------------------------------
КАК ПОЛЬЗОВАТЬСЯ
--------------------------------------------------------
  [WINDOWS]
    1. Если включён VPN — выключите его.

    2. Нажмите «Запустить». Подождите 1–3 минуты, пока статус не станет
       «Готово». Задержка на ~10% с надписью «Поиск моста» — это
       нормально, идёт подбор рабочего узла Snowflake. Просто ждите.

    3. Нажмите «Копировать» — адрес 127.0.0.1:9150 попадёт в буфер
       обмена.

    4. Настройте браузер или программу на этот SOCKS5-прокси (см. ниже).

    5. Не закрывайте программу, пока пользуетесь прокси. Окно можно
       свернуть (крестик) — оно уходит в область уведомлений (значок у
       часов), а прокси продолжает работать. Полностью выйти: правый клик
       по значку в трее → «Выход» (или кнопка «Выход» в окне).

  [ANDROID]
    1. Нажмите «Подключить» и разрешите запрос VPN.

    2. Подождите 1–3 минуты, пока не появится «Подключено». Способ
       подключения подбирается автоматически (режим Smart) — настраивать
       ничего не нужно. Не закрывайте приложение в это время.

    3. Нажмите «Выбор приложений» и отметьте те, которым нужен Tor.

    ВАЖНО: через Tor идут только отмеченные приложения. Пока не отмечено
    ни одного — через Tor не идёт ничего. Неотмеченные приложения
    работают через обычный интернет, как и раньше; весь телефон в Tor не
    заворачивается никогда.


--------------------------------------------------------
БРАУЗЕР
--------------------------------------------------------
  [WINDOWS]
    FIREFOX
      Настройки → «Параметры сети» → «Настроить…» →
      «Ручная настройка прокси»:
        • Узел SOCKS:  127.0.0.1
        • Порт:        9150
        • Вариант:     SOCKS 5
        • Включите «Проксировать DNS при использовании SOCKS 5».
      Нажмите ОК.

    CHROME / EDGE
      Удобнее через расширение FoxyProxy:
        • Тип: SOCKS5, адрес 127.0.0.1, порт 9150, включите «Proxy DNS».
      В FoxyProxy можно задать список сайтов — тогда через Tor пойдут
      только заблокированные адреса, а остальное — напрямую и на полной
      скорости.

  [ANDROID]
    Просто отметьте браузер в «Выборе приложений». Внутри браузера
    настраивать прокси НЕ нужно.

    Совет: отметьте отдельный браузер, которым вы не пользуетесь
    постоянно, — остальные останутся на быстром прямом соединении.


--------------------------------------------------------
TELEGRAM
--------------------------------------------------------
  [WINDOWS]
    1. Запустите прокси в программе и дождитесь «Готово».
    2. Telegram Desktop → меню ☰ → «Настройки».
    3. «Продвинутые настройки» (Advanced).
    4. «Тип соединения» → «Использовать свой прокси».
    5. «Добавить прокси» → выберите SOCKS5.
    6. Сервер: 127.0.0.1, Порт: 9150. Логин и пароль оставьте пустыми.
    7. «Сохранить» и убедитесь, что этот прокси выбран.

    Примечание: адрес 127.0.0.1 — это «сам компьютер», поэтому прокси
    виден только с этого ПК. Подключить к нему телефон нельзя — для
    телефона есть отдельное приложение, см. колонку [ANDROID].

  [ANDROID]
    Просто отметьте Telegram в «Выборе приложений». Внутри самого
    Telegram настраивать ничего не нужно — прокси в его настройках
    включать НЕ надо. Если там уже включён свой прокси — выключите его.

    Учтите: через Tor Telegram может загружаться особенно долго — вплоть
    до нескольких минут. Чаты, аватарки и медиа подтягиваются постепенно,
    приложение может выглядеть «зависшим». Это нормально, подождите.


--------------------------------------------------------
ПРОВЕРКА
--------------------------------------------------------
  [WINDOWS]  После статуса «Готово» нажмите «Проверить, что работает
             через Tor» — программа откроет check.torproject.org через
             прокси и покажет, идёт ли трафик через сеть Tor и ваш
             внешний IP.

  [ANDROID]  Откройте в отмеченном браузере check.torproject.org. Если
             написано «Congratulations. This browser is configured to use
             Tor» — всё работает.


--------------------------------------------------------
ЕСЛИ НЕ ПОДКЛЮЧАЕТСЯ
--------------------------------------------------------
  [WINDOWS]
    • Убедитесь, что VPN выключен и есть интернет.
    • Snowflake иногда долго ищет рабочий узел — подождите пару минут или
      нажмите «Остановить», затем «Запустить» снова.
    • В разделе «Дополнительно» можно добавить свои мосты obfs4 (получить
      на https://bridges.torproject.org или в Telegram у @GetBridgesBot),
      нажать «Сохранить мосты» и перезапустить прокси.

  [ANDROID]
    • Выключите другой VPN — Android разрешает работать только одному
      VPN одновременно.
    • Нажмите «Настроить» и выберите «Smart» или «Snowflake AMP».
      Обычный «Snowflake» часто застревает на 10%.
    • Переключитесь между Wi-Fi и мобильным интернетом.
    • Полностью выключите и снова включите подключение.


--------------------------------------------------------
ДОПОЛНИТЕЛЬНО
--------------------------------------------------------
  [WINDOWS]  • «Запускать вместе с Windows» — галочка в разделе
               «Дополнительно».
             • Файлы программы лежат в %LOCALAPPDATA%\TorProxy
               (кнопка «Открыть папку с настройками»).

  [ANDROID]  • Подробная пошаговая инструкция — во вкладке «Инструкция»
               внутри самого приложения.


--------------------------------------------------------
О ПРИВАТНОСТИ
--------------------------------------------------------
  Трафик идёт через сеть Tor: провайдер не видит, какие сайты вы
  открываете, а сайт не видит ваш настоящий IP. Это обход блокировки,
  а не гарантия полной анонимности: не входите в аккаунты, которые не
  хотите связывать между собой.
